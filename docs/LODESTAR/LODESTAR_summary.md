# LODESTAR Trial 요약 (JAMA, 2023)

## 논문 정보
- **제목**: Treat-to-Target or High-Intensity Statin in Patients With Coronary Artery Disease
- **저널**: JAMA, 2023
- **저자**: Hong SJ, Lee YJ 등 (연세대 세브란스병원 심장내과 주도)
- **등록**: ClinicalTrials.gov NCT02579499

---

## 연구 배경

- CAD 환자에서 LDL-C를 낮추기 위한 스타틴 처방 전략이 두 가지 있음:
  - **고강도 스타틴 전략**: 처음부터 고강도 스타틴을 쓰고 목표 수치 없이 유지 (AHA/ACC 권고)
  - **Treat-to-target 전략**: 중강도 스타틴으로 시작 → LDL-C 수치를 보면서 용량 조절 (ESC/EAS 권고)
- 두 전략을 **직접 비교한 RCT가 없었음**
- 고강도 스타틴은 단순하지만, 개인별 약물 반응 차이와 장기 부작용 우려가 있음
- Treat-to-target은 맞춤형 접근이 가능하지만, RCT 근거가 부족했음

---

## 연구 설계

| 항목 | 내용 |
|------|------|
| 디자인 | 다기관, 무작위, 개방형, **비열등성** 시험 |
| 기관 | 한국 12개 기관 |
| 대상 | CAD 환자 **4,400명** |
| 배정 | 1:1 (LDL-C ≥100, ACS 여부, 당뇨로 층화) |
| Treat-to-target군 (n=2,200) | LDL-C **50-70 mg/dL** 목표로 스타틴 강도 조절 |
| 고강도 스타틴군 (n=2,200) | Rosuvastatin 20mg 또는 Atorvastatin 40mg 고정 |
| 비열등성 마진 | **3.0%** |
| 추적 기간 | 3년, **98.7% 추적 완료** |

### 대상 환자 특성
- 평균 나이 65세, 남성 72%, 고혈압 67%, 당뇨 33%
- 74%는 진단/재관류 후 1년 이상 경과한 안정기 환자
- 기저 LDL-C: 약 86-87 mg/dL
- 무작위 배정 전: 고강도 스타틴 25%, 중강도 스타틴 57%

### Treat-to-target 전략 상세
- 스타틴 미사용자: 중강도 스타틴으로 시작
- 이미 복용 중: LDL-C <70이면 유지, ≥70이면 증량
- 추적 중: LDL-C ≥70 → 증량, 50-70 → 유지, <50 → 감량
- Ezetimibe 병합은 적극 권고하지 않음 (스타틴 강도 전략에 집중하기 위해)

### Primary Endpoint
- 3년 복합: **전사망 + MI + 뇌졸중 + 관상동맥 재관류**

### Secondary Endpoints
- 신규 당뇨, 심부전 입원, DVT/PE, 말초혈관 재관류, 대동맥 시술, 말기 신장병, 스타틴 불내성 중단, 백내장 수술, 검사 이상 복합

### Sample Size
- 기존 연구 기반 3년 이벤트율 12% 가정
- 비열등성 마진 3.0%, 단측 α 2.5%, 검정력 80%
- 탈락률 15% 및 스타틴 종류(rosuvastatin/atorvastatin) 균형 고려
- **최종 4,400명 (각 군 2,200명)**

### 통계 분석
- Primary: As-randomized population, 비열등성 검정
- Com-Nougue approach (Kaplan-Meier failure rate + Greenwood formula)
- 97.5% CI 상한 < 3.0%이면 비열등성 선언
- Per-protocol 민감도 분석

## 주요 결과

### Primary Endpoint (3년 복합 종말점)

| | Treat-to-target군 | 고강도 스타틴군 | 절대 차이 |
|---|---|---|---|
| 발생률 | **8.1%** (177명) | **8.7%** (190명) | **-0.6%** (97.5% CI 상한: 1.1%) |

- **비열등성 충족** (97.5% CI 상한 1.1% < 마진 3.0%, P<.001)
- Per-protocol 분석에서도 일관 (8.3% vs 8.5%, 97.5% CI 상한 1.5%)

### 개별 종말점

| 종말점 | Treat-to-target | 고강도 스타틴 | 절대 차이 (95% CI) | p |
|--------|----------------|-------------|-------------------|---|
| 전사망 | 2.5% (54명) | 2.5% (54명) | <0.1% (-0.9~0.9) | .99 |
| MI | 1.6% (34명) | 1.2% (26명) | 0.4% (-0.3~1.1) | .23 |
| 뇌졸중 | 0.8% (17명) | 1.3% (27명) | -0.5% (-1.1~0.1) | .13 |
| 관상동맥 재관류 | 5.2% (112명) | 5.3% (114명) | -0.1% (-1.4~1.2) | .89 |

### LDL-C 수치
- 전체 기간 평균 LDL-C: **69.1 mg/dL vs 68.4 mg/dL** (P=.21, 유의한 차이 없음)
- 6주 시점에서만 treat-to-target군이 유의하게 높았음 (69.6 vs 66.8, P<.001) → 용량 조절 시간 필요
- 6주 이후에는 차이 없음
- Treat-to-target군의 LDL-C <70 달성률: 56%(1년), 61%(2년), 58%(3년)

### 스타틴 사용 패턴
- Treat-to-target군: 고강도 **54%**, 중강도 **43%** 사용
- 고강도 스타틴군: 고강도 **92%** 사용
- Treat-to-target군에서 증량 17%, 감량 9%, 유지 73%
- Ezetimibe 사용: treat-to-target 20% vs 고강도 11% (3년 시점)

### 안전성 (이차 결과)

| 항목 | Treat-to-target | 고강도 스타틴 | 절대 차이 (95% CI) | p |
|------|----------------|-------------|-------------------|---|
| 신규 당뇨 | 5.6% | 7.0% | -1.3% (-2.8~0.1) | .07 |
| 스타틴 중단 | 1.5% | 2.2% | -0.7% (-1.5~0.1) | .09 |
| 검사 이상 복합 | 0.8% | 1.3% | -0.5% (-1.1~0.1) | .11 |
| 말기 신장병 | 0.1% | 0.5% | -0.3% (-0.7~0.0) | .05 |
| 백내장 수술 | 2.0% | 1.9% | 0.1% (-0.8~0.9) | .90 |
| 심부전 입원 | 0.6% | 0.3% | 0.3% (-0.1~0.7) | .17 |
| **신규당뇨+검사이상+말기신장병 복합 (post-hoc)** | **6.1%** | **8.2%** | **-2.1% (-3.6~-0.5)** | **.009** |

- NNH = 48 → 고강도 스타틴군이 안전성 면에서 불리

### 하위군 분석
- 나이, 성별, BMI, 고혈압, 당뇨, CKD, 임상 양상, 기저 LDL-C 등 **모든 하위군에서 일관된 비열등성**

---

## 제한점

1. **개방형(open-label)** 디자인 (단, 맹검 위원회가 사건 판정)
2. 예상보다 낮은 이벤트 발생률 → 마진 3.0%가 넓을 수 있음
3. 개별 종말점 비교 어려움 (이벤트 수 적음)
4. CAD 환자만 포함 → 1차 예방 환자에 일반화 어려움
5. Treat-to-target군의 LDL-C 목표 달성률이 **약 60%로 낮음** (ezetimibe 병용이 적었기 때문)
6. 3년 추적은 장기 효과 반영에 부족할 수 있음

---

## 결론

CAD 환자에서 **LDL-C 50-70 mg/dL을 목표로 스타틴 강도를 조절하는 treat-to-target 전략**은 고강도 스타틴 고정 전략 대비:

- 3년 심혈관 복합 종말점에서 **비열등**
- 고강도 스타틴 사용이 **절반 수준**으로 줄어듦
- 안전성 복합 지표에서 **유의하게 유리** (post-hoc)

→ 모든 환자에게 일률적으로 고강도 스타틴을 쓰는 것보다, **개인 반응을 보면서 용량을 조절하는 맞춤형 전략**이 충분히 효과적이고 안전할 수 있음을 시사

## RACING과의 비교

| | RACING (Lancet 2022) | LODESTAR (JAMA 2023) |
|---|---|---|
| 대상 | ASCVD 환자 3,780명 | CAD 환자 4,400명 |
| 비교 | 중강도 스타틴+ezetimibe vs 고강도 스타틴 | Treat-to-target vs 고강도 스타틴 |
| 핵심 질문 | **약물 병합**으로 스타틴 감량 가능? | **목표 기반 조절**로 스타틴 감량 가능? |
| Primary endpoint | CV death + major CV event + 비치명적 뇌졸중 | 전사망 + MI + 뇌졸중 + 관상동맥 재관류 |
| 비열등성 마진 | 2.0% | 3.0% |
| 결과 | 9.1% vs 9.9% (비열등) | 8.1% vs 8.7% (비열등) |
| LDL-C | 병합군이 유의하게 낮음 (58 vs 66) | 두 군 차이 없음 (69.1 vs 68.4) |
| 안전성 강점 | 불내성 중단 유의하게 적음 | 안전성 복합 유의하게 적음 (post-hoc) |
| 결론 | 비열등 + LDL-C 목표 달성 우월 | 비열등 + 맞춤형 접근 가능 |
| 연구팀 | 세브란스 심장내과 (홍명기 교수) | 동일 |
